# ⚔ GLADIUS 新手指南

---

## 基本概念

你是一名角鬥士，透過戰鬥賺取金幣和經驗值，收集裝備，不斷變強。

**介面分六個頁籤：**

| 頁籤 | 用途 |
|------|------|
| 地下城 | 主要戰鬥內容 |
| 競技場 | 挑戰其他玩家掠奪金幣 |
| 任務 | 領取每日/每週/成就獎勵 |
| 商店 | 購買裝備、競標稀有品 |
| 背包 | 管理裝備、出售物品 |
| ⚒ 鍛造 | 強化裝備、訓練屬性 |

---

## 第一步：打怪練等

進入**地下城 → 探險**，選等級最低的關卡開始。

探險是單怪速戰，沒有風險，適合快速練等和賺錢。打贏自動結算，不需要任何操作。

等級夠了再挑戰下方的**副本**（三波怪物＋Boss），獎勵更豐厚。

---

## 裝備與顏色

裝備有五個品質等級，顏色越亮越強：

| 顏色 | 品質 |
|------|------|
| 灰白 | 普通 |
| 🟢 綠 | 魔法 |
| 🔵 藍 | 稀有 |
| 🟣 紫 | 傳說 |
| 🟠 橙 | 神話 |

打怪有機率掉落裝備，彈出視窗時可以選擇**直接裝備**、**放入背包**或**丟棄**。

---

## 武器怎麼選

每種武器有獨特特性，選適合自己打法的：

| 武器 | 特性 | 適合情境 |
|------|------|---------|
| 斧 | 無視 20% 防禦 | 打高防敵人、Boss |
| 長矛 | 永遠先手攻擊 | 穩定輸出 |
| 匕首 | 首回合傷害 ×1.5 | 快速終結弱敵 |
| 法杖 | 吸血效果翻倍 | 搭配吸血詞條 |
| 鐮刀 | 爆擊率 +10% | 搭配爆擊裝備 |
| 三叉戟 | 流血每回合額外傷 | 持久戰 |

**建議新手**：先用斧，無視防禦讓傷害穩定。

## 金幣管理

**死亡會扣金幣**，但有保護機制：
- 最多扣 300 金幣
- 金幣永遠不會低於 50

**保持金幣正收益的方式：**
1. 探險永遠能賺錢，打不過副本就刷探險
2. 背包裡的普通裝備賣掉，不要囤著
3. 商店可以買到等級相符的裝備，但不急著花大錢

---

## 鍛造系統

裝備升級有兩種方式：

**強化（+1 到 +10）**
- 選中裝備點強化，花費金幣嘗試
- 成功率從 +1 的 90% 降到 +10 的 8%
- +1 到 +3 失敗維持不降，+4 以上失敗降一級
- **金幣只在嘗試時扣，不會雙重懲罰**

**訓練屬性**
- 花金幣永久提升攻擊、防禦、HP、速度
- 費用隨等級和訓練次數遞增
- 訓練後金幣不會低於 50（保護機制）

---

## 競技場

進入**競技場**，對戰隨機生成的對手：

- 🟢 **較弱**：容易贏，掠奪較少
- 🔵 **相當**：五五波
- 🟠 **較強**：難度高，獎勵豐厚

**勝利**：掠奪對手攜帶金幣的 10-25%  
**失敗**：受傷休息 30 分鐘，損失少量金幣

每天 5 次免費刷新對手，也可花金幣額外刷新。

---

## 傭兵副本

從探險和副本掉落**傭兵契約捲軸**，在**背包**選取後去**地下城 → 傭兵副本**出發。

傭兵會幫你打怪，高防傭兵擋傷，治癒師回血。捲軸使用後消耗。

---

## 任務系統

每天記得去**任務**頁籤看看，分三類：

- **每日任務**：簡單目標，每天重置
- **每週任務**：較高要求，獎勵稀有裝備
- **成就**：長期里程碑，完成後永久鎖定

頁籤上有紅色數字提示可領取的任務數。

## 新手推薦路線

```
Lv.1-5   → 刷探險：狼群獵場、山賊巢穴
Lv.5-10  → 副本：野狼森林普通難度
Lv.10-15 → 開始逛商店競標欄，搶稀有裝備
Lv.15+   → 挑戰副本英雄難度，刷傳說/神話裝備
任何時候  → 記得每天領任務獎勵
```

## 常見問題

**打不過 Boss 怎麼辦？**  
先刷探險把等級拉高，或換斧（無視防禦）。Boss 建議等級超過副本要求 3-5 級再挑戰。

**金幣不夠怎麼辦？**  
刷探險（穩定賺錢）+ 賣掉背包普通裝備 + 競技場打較弱對手。

**裝備什麼時候換？**  
攻擊或防禦有明顯提升就換，別留戀普通品質的舊裝備。
